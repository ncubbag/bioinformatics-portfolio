# Sensory Integration Database (SQL Project)

This project models how human sensory systems interact, capturing
stimuli, measurements, and responses across multiple modalities
(auditory, visual, nociceptive, gustatory, etc.).

## Features
- Fully normalized relational schema
- ERD with crow’s-foot notation
- DDL for tables, indexes, views, and triggers
- Seed data for testing
- 20+ business-annotated queries:
  - Multi-table joins
  - Aggregations with GROUP BY / HAVING
  - Ordered result sets
  - Limited outputs

## Structure
- `schema/` – Table creation scripts
- `data/` – Sample INSERT statements
- `queries/` – Analytical and reporting SQL
- `erd/` – Conceptual data model

## Highlights
- Demonstrates real-world schema design
- Includes indexing strategy
- Uses views and triggers
- Emphasizes business logic in comments
